# Install and Run

```bash
git clone https://github.com/JLee21/react-graphql-geopins.git
cd react-graphql-geopins

# Create MongoDB URI and Google's OAuth Client ID inside a file .env
MONGO_URI=<goes here>
OAUTH_CLIENT_ID=<goes here>

yarn install
yarn dev

cd client
yarn install
yarn start
```
